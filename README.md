##E-Commerce Website
Introduction:
This project is about building an e-commerce website using Spring Boot for the backend,
Thymeleaf for the frontend, and PostgreSQL for the database. The goal is to make a website
where customers can browse products, search easily, add items to their cart without logging in,
and only log in when they actually want to buy something. Orders and customer details will be
saved in the database, and admins will have their own dashboard to manage products and
orders.
